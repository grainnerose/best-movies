## Getting Started

1. `bundle install`
2. `bundle exec shotgun -p 3000 -o 0.0.0.0` or `bundle exec shotgun -p $PORT -o $IP` on Cloud9
3. Visit `http://localhost:3000/` in your browser or share/application on Cloud9

## Starting Tux

1. `bundle exec tux -r ./config/environment.rb`
